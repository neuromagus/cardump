### Cardump (cars auction)

Used technologies:
- Dotnet microservices
- Postgres/MongoDB
- RabbitMQ
- Duende Identity server
- SignalR
- gPRC
- NextJS with Tailwind
- Docker
- Nginx-proxy
- Gateway using Microsoft YARP
